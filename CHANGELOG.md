# 更新日志

所有版本更新记录。

## v1.0.0 (2026-06-06)

首个版本发布。

### 新增功能
- **AI 文章生成**：根据话题、难度、关键词生成英语阅读文章
- **6 级难度**：四级/六级/考研 × 易/难
- **12 个预设话题**：科技、新闻、教育、健康、环境、经济、文化、体育、旅行、日常生活、职场、心理学
- **点击查词**：单击任意单词显示翻译、音标、词性
- **段落翻译**：每段英文后有「译」按钮，点击查看中文翻译
- **全文翻译**：一键显示/隐藏所有段落的中文翻译
- **句子分析**：长按选中句子，AI 分析语法结构、翻译、仿写练习
- **单词发音**：真人发音（Free Dictionary API）+ Google TTS + 设备 TTS 三级降级
- **生词本**：收藏单词，支持复习卡片模式
- **学习词库**：导入学过的单词，AI 生成文章时自动融入
- **智能复习**：生成文章时自动使用学习词库中的单词
- **词根去重**：running/runs/ran 自动识别为同一词根
- **暗黑模式**：支持亮色/暗色主题切换
- **对话历史**：聊天记录自动保存，切换页面不丢失
- **单词导入**：支持手动粘贴或 PDF 解析导入单词
- **本地词典**：内置 19000+ 高频词，无需联网即可查词

### 技术实现
- 纯前端架构：HTML + CSS + JavaScript
- 数据存储：IndexedDB（浏览器内置）
- AI API：DeepSeek（兼容 OpenAI 协议）
- 打包框架：Capacitor → Android APK
- 词典数据：ECDICT 精简版（1.1MB）
