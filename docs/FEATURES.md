## AI 文章生成

### 功能说明

根据用户输入的话题、难度、关键词，使用 DeepSeek API 生成英语阅读文章，同时生成对应的中文翻译。

### 使用方法

1. 进入「对话」页面
2. 选择难度（四级/六级/考研）
3. 选择话题（预设或自定义）
4. 输入具体要求（如"写一篇关于人工智能的文章"）
5. 点击「生成」按钮
6. 等待 AI 生成文章
7. 点击「阅读全文」进入阅读页面

### 难度说明

| 等级 | 词汇量 | 句式特点 |
|------|--------|----------|
| 四级（易） | 3000 词 | 简单句为主 |
| 四级（难） | 4500 词 | 复合句为主 |
| 六级（易） | 5000 词 | 复合句+少量嵌套 |
| 六级（难） | 6000 词 | 长难句+多层嵌套 |
| 考研（易） | 5000 词 | 长难句，同位语 |
| 考研（难） | 5500 词 | 多重嵌套，熟词僻义 |

### 预设话题

科技、新闻、教育、健康、环境、经济、文化、体育、旅行、日常生活、职场、心理学

## 点击查词

### 功能说明

在阅读页面，单击任意英文单词，弹出翻译窗口，显示音标、词性、中文翻译。

### 使用方法

1. 进入阅读页面
2. 单击任意英文单词
3. 弹出翻译窗口
4. 点击「🔊」按钮播放发音
5. 点击「+ 收藏」将单词加入生词本

### 词典来源

1. **本地词典**：19000+ 高频词（离线可用）
2. **Free Dictionary API**：在线查询（需联网）
3. **AI 翻译**：兜底方案（需联网 + 消耗 API 额度）

### 词根识别

自动识别单词变体：
- running → run
- experiences → experience
- cities → city
- better → good

## 段落翻译

### 功能说明

每段英文文章后有一个「译」按钮，点击显示该段落的中文翻译。

### 使用方法

1. 进入阅读页面
2. 每段英文下方有「译」按钮
3. 点击「译」→ 显示中文翻译，按钮变为「隐」
4. 点击「隐」→ 隐藏翻译
5. 顶栏「显示全部翻译」按钮可一次性显示所有翻译

## 句子分析

### 功能说明

长按选中英文句子，AI 分析语法结构、翻译、重点词汇、仿写练习。

### 使用方法

1. 进入阅读页面
2. 长按并拖动选中一个句子
3. 弹出「问 AI」按钮
4. 点击按钮，等待 AI 分析
5. 弹窗显示：
   - 原句
   - 翻译（直译 + 意译）
   - 语法结构分析
   - 重点词汇/短语
   - 仿写练习

## 单词发音

### 功能说明

点击单词翻译窗口中的「🔊」按钮，播放真人发音。

### 发音来源（按优先级）

1. **真人录音**：Free Dictionary API 提供的 MP3 音频（需联网）
2. **Google TTS**：Google 翻译的语音合成（需联网）
3. **设备 TTS**：Android 系统自带的语音合成（离线可用）

### 使用方法

1. 单击单词，弹出翻译窗口
2. 点击「🔊」按钮
3. 等待发音播放

## 生词本

### 功能说明

收藏不认识的单词，方便后续复习。

### 使用方法

1. **收藏单词**：在阅读页面，单击单词 → 点击「+ 收藏」
2. **查看生词本**：点击导航栏「生词本」
3. **删除单词**：点击单词右侧的「移除」按钮
4. **开始复习**：点击「开始复习」进入闪卡模式

### 自动同步

收藏单词时，会自动同步到学习词库，用于生成文章时融入。

## 学习词库

### 功能说明

导入学过的单词，AI 生成文章时会自动使用这些单词，帮助巩固记忆。

### 使用方法

1. **导入单词**：
   - 在对话页面点击「导入单词」
   - 选择导入方式（手动粘贴 / PDF 文件）
   - 粘贴单词列表或上传 PDF
   - 点击「导入」
2. **查看词库**：点击「学习词库」
3. **管理模式**：点击「管理」按钮
   - 可删除单个单词
   - 可清空全部

### 智能复习

生成文章时，自动从学习词库抽取 8 个单词加入关键词，AI 会在文章中自然使用这些单词。

### 词根去重

自动识别单词变体，避免重复：
- running / runs / ran → run
- cities / city → city

## 复习卡片

### 功能说明

闪卡模式复习生词，正面显示英文，背面显示中文。

### 使用方法

1. 进入「生词本」页面
2. 点击「开始复习」
3. 看到英文单词 + 音标（正面）
4. 点击卡片翻转，显示中文翻译（背面）
5. 点击「记住了」→ 移除该词
6. 点击「继续」→ 保留该词
7. 复习完成显示统计

## 单词导入

### 功能说明

批量导入学过的单词到学习词库。

### 导入方式

#### 手动粘贴
1. 选择「手动粘贴」
2. 在文本框中粘贴单词（每行一个，或用逗号/空格分隔）
3. 点击「导入」

#### PDF 文件
1. 选择「PDF 文件」
2. 点击选择文件
3. 等待解析完成
4. 点击「导入」

### 支持格式

- 每行一个单词
- 逗号分隔
- 空格分隔
- 混合格式

## 暗黑模式

### 功能说明

切换亮色/暗色主题，保护眼睛。

### 使用方法

1. 点击导航栏右侧「🌙」按钮
2. 切换为暗黑模式
3. 再次点击切换回亮色模式
4. 主题设置自动保存

## 设置页面

### 功能说明

管理应用配置，包括难度模式、API 设置、外观。

### 设置项

#### 难度模式
- **易**：基础难度，适合入门和巩固
- **难**：接近真题难度，适合冲刺备考

#### 外观
- 亮色模式
- 暗黑模式

#### API 设置
- **API Key**：DeepSeek API 密钥
- **Base URL**：API 地址（默认 https://api.deepseek.com/v1）
- **模型**：DeepSeek V4 Flash / Pro / 自定义

### 使用方法

1. 点击导航栏右侧「⚙️」按钮
2. 修改设置
3. 点击「保存设置」

## 技术说明

### 数据存储

- **localStorage**：API Key、主题、对话历史
- **IndexedDB**：文章、生词本、学习词库

### 离线功能

- 本地词典查询（19000+ 词）
- 已保存文章的阅读
- 生词本查看和复习
- 暗黑模式切换

### 需要联网

- AI 文章生成
- AI 句子分析
- 在线词典查询
- 单词发音（真人录音 / Google TTS）
- PDF 解析（使用 pdf.js CDN）
