## 功能一：语境例句生成

### 目标
点击单词后，tooltip 中显示 2-3 个不同语境的例句。

### 数据源
1. Free Dictionary API — 部分单词有 `example` 字段（免费，无延迟）
2. AI 生成 — 无例句时调用 DeepSeek（有延迟，缓存结果）

### Tooltip 扩展

```
┌─────────────────────────────┐
│ elaborate                    │
│ [ɪˈlæbərət]  adj.           │
│ 精心制作的；详尽的            │
│                              │
│ 📝 例句：                     │
│ • He gave an elaborate       │
│   explanation.（日常）         │
│ • The report was elaborate   │
│   and well-researched.（学术）│
│                              │
│ 四级  高频  [+ 收藏] [🔊]     │
└─────────────────────────────┘
```

### 实现

```javascript
// dictionary.js — 从 API 响应提取例句
function extractExamples(apiResponse) {
  const examples = [];
  for (const meaning of apiResponse.meanings || []) {
    for (const def of meaning.definitions || []) {
      if (def.example) {
        examples.push({ sentence: def.example, pos: meaning.partOfSpeech });
      }
    }
  }
  return examples.slice(0, 3);
}

// api.js — AI 兜底生成
async function generateExamples(word, level) {
  const prompt = `为"${word}"生成3个不同语境的例句（日常/学术/商务），难度${level}，每句10-20词。返回JSON：{"examples":["..."]}`;
}
```

### 缓存策略
- 例句存入 localStorage（key: `examples_${word}`）
- 有效期 30 天
- 已缓存直接显示，无网络也能用

### 改动文件
- `js/dictionary.js` — 提取例句 + 缓存
- `js/api.js` — 新增 generateExamples 方法
- `js/components/tooltip.js` — 显示例句区域
- `css/style.css` — 例句样式

## 功能三：阅读报告

### 目标
每周/月自动生成阅读报告，包含阅读量、词汇增长、速度趋势、成就。

### 报告内容

```
┌─────────────────────────────────────┐
│  📊 本周阅读报告 (6.1 - 6.7)        │
│                                     │
│  📖 阅读量                          │
│  ├── 5 篇文章 · 2,100 词            │
│  ├── 总阅读时间：45 分钟             │
│  └── 平均速度：180 词/分             │
│                                     │
│  📚 词汇                            │
│  ├── 新学：45 个                    │
│  ├── 复习：120 个                   │
│  └── 掌握：30 个（间隔>21天）        │
│                                     │
│  🔥 连续：7 天                      │
│                                     │
│  🏆 成就                            │
│  ✅ 初次阅读  ✅ 百词达人            │
│  ✅ 连续 7 天  ⬜ 连续 30 天         │
│                                     │
│  [分享] [关闭]                       │
└─────────────────────────────────────┘
```

### 成就系统

| 成就 | 条件 | 图标 |
|------|------|------|
| 初次阅读 | 完成第 1 篇计时阅读 | 📖 |
| 百词达人 | 学习词库达到 100 词 | 📚 |
| 千词大师 | 学习词库达到 1000 词 | 🎓 |
| 连续 3 天 | 连续阅读 3 天 | 🔥 |
| 连续 7 天 | 连续阅读 7 天 | 🔥🔥 |
| 连续 30 天 | 连续阅读 30 天 | 🔥🔥🔥 |
| 速度之星 | 单次阅读 > 250 词/分 | ⚡ |
| 阅读马拉松 | 单次阅读 > 10 分钟 | 🏃 |

### 数据来源
- readingStats 表 → 阅读量、时间、速度
- learnWords 表 → 词汇统计
- articles 表 → 文章数

### 触发方式
- 统计页面添加「生成报告」按钮
- 每周首次打开 App 自动弹出（可选）

### 改动文件
- `js/views/report.js` — 新增，报告页面
- `js/views/stats.js` — 添加报告入口
- `js/router.js` — 添加路由
